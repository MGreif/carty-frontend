# PartyCarty-Frontend

This Project was realised using the Next.JS SSR Framework.

# Installation

- Install dependencies using `npm install`

- create .env file with following variables:

|Name|Value|
|-----|-----|
|`NEXT_PUBLIC_SERVICE_URI`|URI of the party-carty service|
